---

GriMoire is a SharePoint Framework web part that combines an LLM assistant with a visual block system, intent-based search, and Microsoft 365 actions. The Hybrid Interaction Engine (HIE) keeps the model grounded in the UI it creates — interactions with blocks feed back into the conversation as structured events.
